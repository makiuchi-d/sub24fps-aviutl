-----------------------------------------------------------------------
      自動24fps(代理) フィルタプラグイン ver0.01  by MakKi
-----------------------------------------------------------------------

【機能】

　AviUtl内蔵の自動24fpsインターレース解除フィルタを
  通常のビデオフィルタとして使ってしまうフィルタです。

【インストール】

　sub24fps.auf を AviUtl の認識可能なフォルダに入れてください

【使用方法】

  全てのパラメータはAviUtl0.98dおよび0.99内蔵の自動24fpsと共通です。
  AviUtl0.99で使用する場合はドット単位解除は無視されます。

【注意事項】

  AviUtl 0.98d 以降専用(0.98b以前のバージョンでは起動直後に異常終了します)
  このプログラムはフリーソフトウェアです。
  このプログラムによって損害を負った場合でも、作者は責任を負いません。
